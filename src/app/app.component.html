About this app

This app was created to learn the basics in Angular and as a repeat practice to use a simple Microservices REST backend written in Java.

The design of the app and it's use is a combination of the current Burton Snowboards website and the boardfinder that could be found on Burton's website some years ago. On purpose to be able to create the app in relatively short time it has been kept very simple and limited.

Due to being unable to load all the snowboard data from Burton Snowboards via an API the boardfinder app is in it's current state limited to some but not all Burton Men's snowboards.
